CSV Schema

Header (column names):
article, actor, aim, aim_category, deontic, condition, method

Examples are made up. 

Row example:
article, actor, aim, aim_category, deontic, active condition, method
1, "marszalek", "wskazac", "wybiera", "musi", "w okresie 30 dni o poczatku kadencji Sejmu", ""

Matrices should be created for EACH aim_category. 
Aim matrix defines the particular aim (e.g. "wskazuje" for aim_category "powołuje")
Condition, Deontic and Method matrix are for describing relations of a particular aim_category. 

Matrix examples for aim_category = "powołuje"

Note that Condition, Deontic and Method matrices CANNOT have values in cells where Aim Matrix is empty. 
But they may not have value in cells where Aim Matrix is non-empty.
